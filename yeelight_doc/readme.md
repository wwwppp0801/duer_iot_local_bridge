## 协议示例

### 开关

```javascript
{"id":1,"method":"toggle"}
```

### 调亮度

```javascript
{"id":1,"method":"set_bright","params":[40]};
```


### 打开夜灯模式

```javascript
{"id":1,"method":"set_scene", "params": ["nightlight", 100]}
```
